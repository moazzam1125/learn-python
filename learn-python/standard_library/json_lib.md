# JSON spec

## Serialization

| Python | JSON |
|--------|------|
| dict | object |
| list, tuple | array |
| str |	string |
| int, long, float | number |
| True  | true |
| False | false |
| None | null |

## Deserialization

| JSON | Python |
|------|--------|
| object | dict |
| array | list |
| string | str |
| number (int) | int |
| number (real) | float |
| true | True |
| false | False |
| null | None |
